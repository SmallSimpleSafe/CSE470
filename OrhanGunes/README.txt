### OrhanGüneþ folder

shows the use of (https://github.com/SmallSimpleSafe/CSE470/tree/master/OrhanGüneþ) for Ellipse2D in java

The Ellipse2D class describes an ellipse that is defined by a framing rectangle.
This class is only the abstract superclass for all objects which store a 2D ellipse. The actual storage representation of the coordinates is left to the subclass.

Here are the files in this folder:

* `OrhanEllipse2D.jar` jar files  --run the project

* `Teacher.txt` teacher file --learn how to work MyEllipse2D.java file and learning "java.awt.geom.Ellipse2D" class

* `MyEllipse2D.java` java file --writing java code

Reference: https://docs.oracle.com/javase/7/docs/api/java/awt/geom/Ellipse2D.html
